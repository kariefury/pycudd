Welcome to Pycudd!

These are the simple instructions for the setting up Pycudd. They assume you just created a Google Cloud Service VM configured as 64-Bit Ubuntu 14.04

SIMPLIFIED INSTALLATION DIRECTIONS FOR PYCUDD

INSTALLATION: 1. install necessary packages ''' sudo apt-get install git sudo apt-get install make sudo apt-get install gcc sudo apt-get install swig sudo apt-get install g++ sudo apt-get install python-dev '''

  1. Clone the GIT repo ''' git clone https://github.com/kariefury/pycudd.git '''

  2. Build the project ''' cd ~ cd pycudd/cudd-2.4.2/ make cd ~ cd pycudd/pycudd make depend make '''

  3. Edit environment variable PYTHONPATH to include pycudd directory ''' export PYTHONPATH=$PYTHONPATH:pycudd/ '''
